# Projeto de Linguagem de Programação III (CET-078)

## Decisões de Design:

Cliente:

- Nome de Usuário;
- Senha;

Fornecedor:

- Nome da Empresa;
- Categoria do produto fornecido;

Categoria:

Classe de enumeração onde estão dispostas as categorias dos produtos.

Produto:

- Nome do produto;
- Categoria;
- Peso;
- Altura;
- Largura;
- Valor unitário;

Carrinho:

- Lista de produtos;
- Valor total da compra;

Loja:

- Carrinho;
- Lista de clientes;
- Lista de produtos;
- Lista de fornecedores;
